# Nishant - Full-Stack Developer Portfolio

A fast, responsive portfolio focused on production work, engineering outcomes,
and recruiter-friendly project case studies.

## Featured projects

- GrowthHub - mentorship marketplace with booking and Razorpay payments
- HireFlow - AI-assisted job search and resume workspace
- SyncScribe - real-time collaborative document editor

Each featured project includes a live product link, source repository, concise
technical summary, and an optimized preview image.

## Technology

- React
- TypeScript
- Vite
- CSS
- React Icons

The portfolio intentionally avoids blocking loaders, WebGL, and heavy animation
libraries so its content remains accessible and reliable across devices.

## Local development

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```
